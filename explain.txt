webpack dev server 的作用就是，将建立向xammp那样建立一个本地服务器，
可以实时查看页面更新。不需要借助vscode 的插件来查看页面的更新。


１、npm install webpack-dev-server@3.2.1
npm install webpack-dev-server --save-dev



２、添加 devServer

            const path = require('path');
            module.exports = {
            entry: './src/index.js',
            output: {
                path: path.resolve(__dirname, 'dist/assets'),
                filename: 'bundle.js'
            },

            devServer:{
                //告诉web dev server 要将哪一个文件夹里的index文件显示在浏览器上面。
                contentBase:path.resolve(__direname,"dist"),
                publicPath:"/assets",
            }
            };


publicPath :指定dist文件夹里面的文件夹。
publicPath的意思就是，index.html里绑定的js 文件的文件夹的位置。
distフォルダがサーバーの起点となるディレクトリとし、
その中にassetsフォルダがあり、そこのbundle.jsファイルをindex.htmlから呼ぶかたちです。

3，package.json文件里写上  "serve": "webpack-dev-server"


  "scripts": {
    "babel": "node_modules/.bin/babel ./src/index.js -w -o ./dist/assets/bundle.js",
    "webpack": "node_modules/.bin/webpack -w",
    "serve": "webpack-dev-server"
  },

4,npm run serve
